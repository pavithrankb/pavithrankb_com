# Pavithran KB - Personal Website

A minimal, Apple-style personal portfolio website built with pure HTML, CSS, and JavaScript.

## Features

- Clean, modern design with glassmorphism effects
- Dark/Light mode toggle with system preference detection
- Fully responsive (mobile-first)
- No build step required - pure static files
- Cloudflare Pages ready

## Structure

```
├── index.html    # Main HTML
├── style.css     # Styles
├── script.js     # Theme toggle
└── logo.png      # Site logo
```

## Deployment

Push to GitHub and connect to Cloudflare Pages. No build command needed.
